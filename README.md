# 商人ゲーム制作（Godot 4.3）GitHub スターター

この3ファイルを Godot プロジェクトのルートに置いてから、GitHub Desktop で初回コミット→Publish してください。

## 収録物
- `.gitignore` — Godot 4.x/Mono/IDE/ビルド成果物を除外
- `.gitattributes` — テキスト正規化と Git LFS 設定（画像/音声/3D等）
- `README.md` — 最小の運用メモ

## 初回の流れ（超要約）
1. 3ファイルをプロジェクト直下に配置

2. GitHub Desktop → **Add local repository...** → プロジェクトフォルダ

3. **Commit to main**（Summaryは `initial commit` など）

4. **Publish repository**（Private推奨）

5. Repository Settings → **Git LFS** を有効化
